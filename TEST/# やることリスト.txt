# やることリスト

## (UI刷新版)コーディング

- [ ]  SVG画像をカテゴリーごとに定義
- [ ]  SVG画像の色は服の色に応じて変化するようにする
- ホーム画面
    - [ ]  Yahoo! Open Local Platform (YOLP)で目的地の座標を検索可能にする
    - [ ]  open-meteoから体感温度を取得しコーデ提案機能に組み込む
    - [ ]  YOLPで取得した座標を、open-meteoに渡す処理
    - [ ]  ヘッダー右の”車”トグルを削除
- クローゼット画面
    - [ ]  服のチェックマークと洗濯機アイコンを削除
    - [ ]  ペンアイコン(編集ボタン)、ゴミ箱アイコン(服を捨てる)を追加
    - [ ]  服の名前の下に、”着用回数+1”と、”洗濯かごへ移動”ボタンを追加
- ランドリー画面
    - [ ]  服左のアイコンを削除し、チェックボックスを配置
    - [ ]  ランドリー画面を開いたときは、デフォルトで全ての服のチェックボックスにチェックマークが入っている(洗濯する)が、手動でチェックマークを外したものは洗濯しないようにする。
    - [ ]  “洗濯完了”ボタンは、”☑を洗濯する”というボタンに変更。
- 設定画面
    - [ ]  開発者項目を追加
    - [ ]  開発者項目で、”デバッグ”トグルをONにしたら各種デバッグ用項目が表示される。
